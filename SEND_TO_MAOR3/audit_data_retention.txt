CIS 6.3.2 - Configure Audit Data Retention

6.3.2.1 Ensure audit log storage size is configured
Current State: max_log_file = 
Expected: Site policy (e.g. ≥ 20MB)
Audit Result: FAIL
Remediation: Set max_log_file = <MB> in /etc/audit/auditd.conf

6.3.2.2 Ensure audit logs are not automatically deleted
Current State: max_log_file_action = 
Expected: keep_logs
Audit Result: FAIL
Remediation: Set max_log_file_action = keep_logs

6.3.2.3 Ensure system is disabled when audit logs are full
disk_full_action =  (❌ FAIL)
Remediation: Set disk_full_action = halt OR single
disk_error_action =  (❌ FAIL)
Remediation: Set disk_error_action = halt OR single OR syslog

6.3.2.4 Ensure system warns when audit logs are low on space
space_left_action =  (❌ FAIL)
Remediation: Set space_left_action = email | exec | single | halt
admin_space_left_action =  (❌ FAIL)
Remediation: Set admin_space_left_action = single | halt
